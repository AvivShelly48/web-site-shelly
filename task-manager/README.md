# ניהול משימות משרדי (Task Manager)

אפליקציית Next.js שמרכזת את ניהול המשימות במשרד למערכת אחת — במקום עומס הודעות בקבוצות וואטסאפ.
ניהול משימות יומיות / שבועיות / חודשיות, דשבורד התקדמות, ותזכורות אישיות לכל עובד בוואטסאפ.

## פיצ'רים

- רשימת משימות יומיות / שבועיות / חודשיות עם סטטוס (ממתינה / בתהליך / הושלמה)
- שיוך משימות לעובדים
- **תזכורות וואטסאפ אישיות** — לכל עובד הודעת תזכורת אוטומטית עם המשימות הפתוחות שלו
  (סינון לפי: להיום / השבוע / באיחור), עם שליחה בלחיצה אחת או "שליחה לכל העובדים"
- דשבורד סיכום והתקדמות כללית ולפי עובד
- ניהול רשימת עובדים (שם, טלפון, אימייל)
- הפקת דוח התקדמות ושליחתו:
  - וואטסאפ - יוצר קישור `wa.me` עם הדוח מוכן לשליחה לכל עובד
  - אימייל - שולח את הדוח דרך SMTP (דורש הגדרת משתני סביבה, ראו `.env.example`)

> **הערה על תזכורות:** התזכורות מבוססות קישורי `wa.me` (לחיצה פותחת את וואטסאפ עם ההודעה מוכנה).
> לשליחה אוטומטית מתוזמנת לחלוטין (cron ללא התערבות) נדרש חיבור ל-WhatsApp Business API / ספק כמו Twilio.

## הרצה

```bash
npm install
npm run dev
```

האפליקציה תרוץ בכתובת http://localhost:3000

## אחסון מידע

המידע (עובדים ומשימות) נשמר בקובץ `data/db.json` - מתאים לשימוש מקומי/דמו. לסביבת ייצור מומלץ להחליף לבסיס נתונים.
