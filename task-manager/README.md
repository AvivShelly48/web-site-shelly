# ניהול משימות משרדי (Task Manager)

אפליקציית Next.js לניהול משימות יומיות וחודשיות במשרד, עם דשבורד התקדמות ושליחת דוחות לעובדים בוואטסאפ ובאימייל.

## פיצ'רים

- רשימת משימות יומיות / חודשיות עם סטטוס (ממתינה / בתהליך / הושלמה)
- שיוך משימות לעובדים
- דשבורד סיכום והתקדמות כללית ולפי עובד
- ניהול רשימת עובדים (שם, טלפון, אימייל)
- הפקת דוח התקדמות ושליחתו:
  - וואטסאפ - יוצר קישור `wa.me` עם הדוח מוכן לשליחה לכל עובד
  - אימייל - שולח את הדוח דרך SMTP (דורש הגדרת משתני סביבה, ראו `.env.example`)

## הרצה

```bash
npm install
npm run dev
```

האפליקציה תרוץ בכתובת http://localhost:3000

## אחסון מידע

המידע (עובדים ומשימות) נשמר בקובץ `data/db.json` - מתאים לשימוש מקומי/דמו. לסביבת ייצור מומלץ להחליף לבסיס נתונים.
